Zoho CRM Dashboard API Research:
- Zoho CRM does NOT have REST API for dashboards (confirmed by Zoho support)
- Alternative approach: Use COQL (CRM Object Query Language) to query the underlying data directly
- COQL supports COUNT, GROUP BY, aggregate functions
